🔮 Tarot Reader API

Ultra-compact Node.js edition for Vercel deployment

Generate deterministic tarot spreads using minimal seed strings. No JWT, no encryption - just pure simplicity.

🚀 Quick Start

1. Create a new spread:

GET /api/create-spread
Try it

2. Read cards from spread:

GET /api/spread/{seed}?choose=0,1,2

📖 API Endpoints

  • /api/create-spread - Generate new spread with seed
  • /api/spread/{seed}?choose=0,1,2 - Read specific cards
  • /api/spread/{seed}?choose=0,1,2&format=raw - Pure JSON
  • /api/spread/{seed}?choose=0,1,2&download=true - Download file

⚡ Features

  • 🎯 Ultra-compact URLs - Minimal seed strings
  • 🔄 Deterministic - Same seed = same spread
  • 🌐 Full CORS - Works from any domain
  • 📦 Multiple formats - JSON, download, raw
  • ⚡ Serverless - Vercel optimized
  • 🔓 No auth needed - Public API
